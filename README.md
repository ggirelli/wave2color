# wave2color
How to get a color (as a hexadecimal string) from a wavelength (nm, in the visible range). A function is provided in the following languages:

* Python
* TypeScript
* Javascript

Based on: http://www.efg2.com/Lab/ScienceAndEngineering/Spectra.htm
